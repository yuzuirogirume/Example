# CLAUDE.md

## プロジェクト概要

宮崎市の民泊ビジネスを管理するためのツール群。
Airbnb・Booking.com の予約データを一元管理し、収益の把握・分析・ホームページ運用を行う。

## ファイル構成

```
Example/
├── 収益管理表.html          # 民泊収益管理ダッシュボード（メインアプリ）
├── ホームページ.html         # 物件紹介ホームページ
├── サンプルデータ_Airbnb.csv       # Airbnb の予約データ
├── サンプルデータ_Booking.com.csv  # Booking.com の予約データ
└── README.md
```

## 技術スタック

- HTML / CSS / JavaScript（バニラ、サーバー不要でブラウザのみで動作）
- タスクに応じて Python スクリプトを使ったデータ処理も行う

## Claude への指示

### 作業内容
- HTML/CSS の修正・デザイン改善
- CSV データの読み込み・集計・変換処理
- ダッシュボードへの新機能追加（グラフ表示、自動計算など）
- バグ修正・既存コードの改善

### 守ってほしいルール

1. **大きな変更は事前に確認する** — ファイル全体の構成変更や既存機能の大幅な書き換えを行う前に、必ずユーザーに確認を取ること。
2. **コメントは日本語で書く** — コード内のコメントやドキュメントはすべて日本語で記述すること。
3. **既存のデザインスタイルを尊重する** — HTML/CSS を編集する際は、既存のフォント・カラーパレット・レイアウトのスタイルを壊さないよう注意すること。
4. **CSV ファイルは読み取り専用** — サンプルデータの CSV ファイルは元データとして扱い、直接編集しないこと。
