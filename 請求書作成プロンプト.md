# 請求書作成プロンプト（ユズイロ用）

## 使い方

1. 下の「毎回入力する情報」の `[ ]` の中を今回の案件の内容に書き換える
2. プロンプト全体をコピーして Claude に貼り付ける
3. 出力された請求書をコピーして使う（Word・Googleドキュメントなどに貼り付けてもOK）

---

## あなたの固定情報（一度だけ書き換えてください）

> ※ 以下の `【　】` の中をあなたの実際の情報に書き換えてから保存してください。

| 項目 | 内容 |
|------|------|
| 氏名 / 屋号 | 【ユズイロ】 |
| 住所 | 【例：東京都渋谷区〇〇1-2-3】 |
| 電話番号 | 【例：090-0000-0000】 |
| メールアドレス | 【例：yuzuiro@example.com】 |
| 銀行口座 | 【例：〇〇銀行 〇〇支店 普通 1234567 ユズイロ】 |
| インボイス登録番号 | 【例：T1234567890123】※ 未登録なら「なし」と書く |

## Claude に貼り付けるプロンプト（毎回使う）

```
以下の情報をもとに、日本のビジネス慣習に合った請求書を作成してください。
読みやすいテキスト形式で出力してください。

■ 請求者情報（私）
- 氏名 / 屋号：ユズイロ
- 住所：【あなたの住所】
- 電話番号：【あなたの電話番号】
- メールアドレス：【あなたのメールアドレス】
- 振込先：【あなたの銀行口座情報】
- インボイス登録番号：【登録番号またはなし】

■ 請求先情報（今回のクライアント）
- 会社名：[ 会社名を入力 ]
- 担当者名：[ 担当者名を入力 ] 様

■ 今回の案件内容
- 業務内容：Instagramタイアップ投稿（[ 店舗名 ]のPR）
  ※ フィード投稿 [ 枚数 ] 枚・ストーリーズ [ 回 ] 回
- 料金（税抜）：[ 金額 ] 円
- 消費税（10%）：自動計算してください
- 合計金額（税込）：自動計算してください

■ 請求書の基本情報
- 請求書番号：[ 例：2026-001 ]
- 発行日：[ 例：2026年4月25日 ]
- 支払期限：[ 例：2026年5月31日 ]

上記をもとに、以下の形式で請求書を作成してください：
1. 請求書タイトル・番号・発行日
2. 請求先（会社名・担当者名）
3. 請求者（自分の情報）
4. 業務内容と金額の明細（表形式）
5. 合計金額（税込）
6. 振込先口座
7. 支払期限
8. インボイス登録番号（ある場合）
```

## よくある質問

**Q. 消費税はどうすればいいの？**
→ プロンプトに「自動計算してください」と書いてあるので Claude が計算してくれます。税抜金額だけ入力すればOKです。

**Q. インボイス登録番号がない場合は？**
→「なし」または「（インボイス未登録）」と書けばそのまま請求書に反映されます。

**Q. 請求書番号はどうつければいい？**
→ 「年-通し番号」形式がシンプルでおすすめです（例：2026-001, 2026-002...）。

**Q. 源泉徴収は必要？**
→ フリーランスへの報酬は10.21%の源泉徴収が必要な場合があります。クライアントから「源泉徴収あり」と言われたら、プロンプトに「源泉徴収10.21%を差し引いた振込金額も記載してください」と追加してください。
